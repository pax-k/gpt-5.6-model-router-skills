# Policy attestations

The publisher attests that, for the submitted v0.2.1 skills-only bundle:

- The listing and starter prompts describe the implemented behavior accurately.
- The plugin requests no authentication and contains no MCP server, connector,
  hosted service, analytics, advertising, tracking, or credential collection.
- Local writes are limited to user-invoked task work, managed custom-agent setup,
  reversible router configuration, backups, and task ledgers described in the
  skill and public policies.
- The plugin does not bypass Codex runtime permissions or host/tool approvals.
- It fails closed when a selected model/effort route cannot be enforced and does
  not silently substitute models.
- External, destructive, credentialed, costly, and materially out-of-scope
  actions remain subject to the authority recorded in the task and the active
  Codex runtime.
- The plugin does not impersonate OpenAI and the original logo intentionally
  avoids OpenAI trademarks.
- The release contains no secrets, private rollout identifiers, local absolute
  paths, cached bytecode, or private acceptance artifacts.
- Support, privacy, terms, source, and MIT license information are public.
